# MotoBike Almendralejo

Web estática profesional para presentar a MotoBike Almendralejo / MotoBike Service.

## Archivos principales

- `index.html`: estructura completa de la landing.
- `styles.css`: estilos responsive y apariencia visual.
- `script.js`: filtro de motos y menú móvil.
- `assets/`: imágenes generadas como placeholders comerciales.

## Datos pendientes de confirmar con el cliente

- Logo oficial en buena calidad.
- Horario real.
- Fotos reales del taller, entregas y motos.
- Inventario real de motos nuevas y de segunda mano.
- Precios, kilometraje, años y condiciones de cada moto.
- Enlace exacto de Google Maps si la direccion cambia.
